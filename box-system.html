The Universal Box

One box mold, one color, one production run. The sticker is the only variable — making every new collection or limited drop a sticker print away, not a new box design.

The holographic "?" creates mystery energy even when the customer chose a specific character. The unboxing moment stays magical.

  • Matte dusty pink — on-brand for Nancy
  • Holographic iridescent "?" — premium feel
  • "SQUEEZE ME!" badge — universal brand marker
  • Character pops out the top on open
  • Same box = bulk ordering = lower unit cost
  • New collection = new sticker, not new box

Every Collection, One Box

Same pink box — just swap the collection sticker

The Sticker Sheet

Each collection gets its own sticker design. Printed in bulk sheets, applied during packing. New drop? New sticker. Limited edition? Foil variant.

The sticker carries the collection name, count ("1 of X"), and tiny silhouette hint icons at the bottom — so the buyer knows which set it's from without revealing the exact character.
