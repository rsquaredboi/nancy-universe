## TIPS FOR NANO BANANA 2

1. **Upload the reference image first** — always start with the existing product render
2. **Simple prompts work best** — conversational language, not technical jargon
3. **Fix one issue at a time** — if scale is wrong, say "make the plush smaller" in a follow-up
4. **Start fresh if stuck** — if results diverge after 3-4 corrections, new chat with original reference
5. **Include "mobile optimized"** for any comparison or features graphics
6. **Grid method (2x2)** is fastest for batch — 4 distinct images per prompt
7. **Scale reference**: Standard plush = 25cm / ~10 inches (about the size of a hand)
8. **Consistency**: Keep the same chat for all angles of one character to maintain consistency
9. **Download best results immediately** — don't wait, as regeneration may differ
10. **Aspect ratio**: Use 1:1 for product shots, 16:9 for lifestyle/banner, 4:5 for social
